## 🌟 Features

- **AI Chatbot** — Portfolio-aware assistant powered by Groq + Gemini fallback
- **3D & WebGL** — Immersive visuals with Three.js, React Three Fiber, and GLSL shaders
- **Bilingual (EN/ID)** — Full internationalization via `next-intl`
- **Smart Redirect** — Mobile users auto-redirected to the mobile version
- **Dark / Light mode** — System-preferred with smooth toggle
- **GitHub & WakaTime stats** — Live coding activity and repo stats
- **Performance-first** — Dynamic imports, image optimization, Lenis scroll

## 🚀 Getting Started

```bash
# Clone
git clone https://github.com/ArnabOPP/Portfolio.git
cd Portfolio

# Install
npm install

# Configure environment variables
cp .env.example .env.local
# Fill in your API keys (see below)

# Dev
npm run dev
```

### Environment Variables

```env
NEXT_PUBLIC_GITHUB_USERNAME=ArnabOPP
GITHUB_TOKEN=your_github_pat

WAKATIME_API_KEY=your_wakatime_key

GROQ_API_KEY=your_groq_key
GEMINI_API_KEY=your_gemini_key
```

### Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server (Turbopack) |
| `npm run build` | Production build |
| `npm start` | Serve production build |
| `npm run lint` | ESLint |
